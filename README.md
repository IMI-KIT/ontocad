OntoCAD
=======

Semi-automated Ontology Population from Building Construction Drawings.

OntoCAD allows for a fast segmentation and classification of 2D CAD plans.
It was developed for the purpose of extracting semantic information from building CAD plans, but it can easily be employed for any type of 2D CAD data.
Supported import formats are DXF and OWL-RDF.
